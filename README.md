Sistema de Registro de Inasistencias Docentes:
____________________________________________________________________________

Propósito del Proyecto:

Este proyecto ha sido desarrollado con fines exclusivamente educativos. Su objetivo principal es diseñar un sistema informático que permita registrar y gestionar las inasistencias del personal docente, optimizando el control y la trazabilidad de dichos registros dentro de una institución educativa.

El sistema facilita el ingreso y almacenamiento de información relevante, incluyendo: Nombre y apellido del docente, Cédula de identidad, Turnos afectados por la inasistencia y la Fecha de inicio y finalización del período de inasistencia.

----------------------------------------------------------------------------

Requisitos del Sistema:

Sistema Operativo Compatible: Windows 10 o Windows 11, Linux (Ubuntu 20.04 o superior recomendado)
Requisitos Mínimos de Hardware: Memoria RAM: 250 MB Procesador: Dual Core o superior Espacio en disco: 100 MB libres

----------------------------------------------------------------------------

Funcionalidades Principales: 

Registro y eliminacion de inasistencias docente y la consulta a las inasistencias docente registradas.

----------------------------------------------------------------------------

Objetivos Educativos:

El propósito de este proyecto es fortalecer las competencias técnicas y colaborativas de los participantes en áreas clave de la informática, incluyendo:
Programacion en Apache NetBeans enfocado a objetos.
Gestión de bases de datos.
Trabajo en equipo y gestión de proyectos.
